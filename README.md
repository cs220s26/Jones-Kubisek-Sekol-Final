## Overview
Jeopardy Bot is a discord based application which mimics a single round of the TV show Jeopardy, where players answer trivia facts in the form of a question (the 'facts' will be referred to as the question now on). The game consists of 4 categories each with 4 questions representing various price point values. Players take turns selecting questions to answer, and depending on their answer, the points get added or subtracted to their overall score. After all questions are answered (or when a player decides to terminate the game early), the game ends, and each player is scored according to how many points they have. 
## Development Setup/Execution
To launch the bot, firstly you will need to add your discord token to AWS secretes manager. Then you must set up your AWS ec2 instance and allow access to secretes managr (located in the second dropdown under Advanced Details). Then paste the entire userdata.sh file into the user data box (located at the very bottom of the Advanced Details section).
